# Otras alternativas notables

<v-clicks>

### Semantic Kernel (Microsoft)
- **Lenguaje**: C#, Python, Java (no oficial para JS)
- **Enfoque**: Integración enterprise con Azure
- **Ventaja**: Excelente para ecosistema Microsoft

### Haystack
- **Lenguaje**: Python principalmente
- **Enfoque**: Pipelines NLP y RAG
- **Ventaja**: Muy maduro para búsqueda y QA

### Dust
- **Lenguaje**: TypeScript
- **Enfoque**: Plataforma completa para apps LLM
- **Ventaja**: UI incluida, menos código

### txtai
- **Lenguaje**: Python
- **Enfoque**: Búsqueda semántica ligera
- **Ventaja**: Muy rápido y eficiente

</v-clicks>
