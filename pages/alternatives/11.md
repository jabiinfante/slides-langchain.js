# ¿Cuándo JS/TS es viable para AI?

<v-clicks>

### Casos donde tiene sentido

- Tu equipo es **JS/TS puro** y no quieres añadir Python
- Necesitas **edge computing** (Cloudflare Workers, Vercel Edge)
- Tu aplicación es **web-first** con mucho frontend
- Quieres un **stack unificado** (mismo lenguaje front y back)
