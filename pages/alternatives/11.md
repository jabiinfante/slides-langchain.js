# Comparativa de frameworks

| Framework | Mejor para | Curva aprendizaje | Ecosistema |
|-----------|------------|-------------------|------------|
| **LangChain** | Workflows complejos, agentes | Media-Alta | ⭐⭐⭐⭐⭐ |
| **LlamaIndex** | RAG avanzado, indexación | Media | ⭐⭐⭐⭐ |
| **Vercel AI SDK** | Apps React/Next.js | Baja | ⭐⭐⭐ |
| **OpenAI SDK** | Casos simples, control total | Baja | ⭐⭐⭐⭐⭐ |
| **Anthropic SDK** | Apps con Claude | Baja | ⭐⭐⭐⭐ |

### ¿Cómo elegir?

1. **Empieza simple**: SDK directo si tu caso es básico
2. **Crece según necesidad**: Añade framework cuando lo necesites
3. **No sobre-ingenierices**: Usa la herramienta mínima que resuelva el problema
4. **Considera el equipo**: Familiaridad y experiencia del equipo
