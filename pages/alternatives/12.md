# ¿Puedes combinar frameworks?

<v-clicks>

**¡Sí!** No tienes que elegir solo uno

### Combinaciones comunes

```typescript
// LangChain para agentes + Vercel AI SDK para UI
import { AgentExecutor } from "langchain/agents";
import { streamText } from "ai";

// LlamaIndex para RAG + OpenAI SDK para generación
import { VectorStoreIndex } from "llamaindex";
import OpenAI from "openai";

// LangChain para orchestration + Qdrant directo
import { QdrantClient } from "@qdrant/js-client-rest";
import { ChatOpenAI } from "@langchain/openai";
```

### Recomendaciones

- Usa cada herramienta para lo que hace mejor
- Mantén interfaces limpias entre componentes
- No añadas complejidad innecesaria
- Documenta las decisiones arquitectónicas
