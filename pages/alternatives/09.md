# Recomendación Pragmática

<v-clicks>

| Caso de uso | Recomendación |
|-------------|---------------|
| **Prototipo/MVP** | Vercel AI SDK o API nativa |
| **RAG** | LangChain.js (o Python si puedes) |
| **Agentes complejos** | LangGraph / DeepAgents |
| **App React con chat** | Vercel AI SDK |
| **Control total** | APIs nativas del provider |

<div class="p-3 bg-blue-500/10 rounded mt-2 text-sm">

1. **Empieza simple** → API nativa o Vercel AI
2. **Añade complejidad cuando la necesites** → LangChain/LangGraph
3. **No sobre-ingenierices** → La herramienta mínima que resuelva el problema

</div>

</v-clicks>
