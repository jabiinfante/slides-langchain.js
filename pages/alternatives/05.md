# Vercel AI SDK

SDK moderno y minimalista para aplicaciones AI

<v-clicks>

### Características principales

- **React-first**: Hooks optimizados para React/Next.js
- **Streaming UI**: Excelente soporte para respuestas streaming
- **Type-safe**: TypeScript de primera clase
- **Ligero**: Pocas abstracciones, más control

### ¿Cuándo usar Vercel AI SDK?

✅ Aplicaciones Next.js/React
✅ Necesitas UI streaming
✅ Prefieres código simple y directo
❌ Necesitas RAG complejo
❌ Agentes con múltiples tools

### Instalación

```bash
npm install ai
```
