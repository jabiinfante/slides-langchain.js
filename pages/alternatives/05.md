# Vercel AI SDK - Tools y Agentes

<v-clicks>

### Soporte para function calling

```typescript
import { openai } from '@ai-sdk/openai';
import { generateText, tool } from 'ai';
import { z } from 'zod';

const { text } = await generateText({
  model: openai('gpt-4o'),
  tools: {
    weather: tool({
      description: 'Obtiene el clima',
      parameters: z.object({ city: z.string() }),
      execute: async ({ city }) => ({ temp: 22, condition: 'Soleado' }),
    }),
  },
  maxSteps: 5, // ← Loop automático de agente!
  prompt: '¿Qué tiempo hace en Madrid?',
});
```

### ¿Agentes completos? Sí, pero stateless

| Tiene ✅ | No tiene ❌ |
|----------|-------------|
| Loop automático (`maxSteps`) | Persistencia de estado (threads) |
| Tools custom | Tools built-in (code interpreter) |
| Streaming | Orquestación multi-agente |
