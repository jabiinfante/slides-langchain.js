# Vercel AI SDK

El SDK moderno para aplicaciones AI en JS/TS

<v-clicks>

### Por qué destaca

- **Provider-agnostic**: OpenAI, Anthropic, Google, Mistral, Groq...
- **Streaming nativo**: Pensado para UIs reactivas
- **React-first**: Hooks listos para usar (`useChat`, `useCompletion`)
- **Edge-ready**: Funciona en Vercel, Cloudflare Workers, etc.
- **Ligero**: Pocas abstracciones, código directo

### El SDK que más ha crecido en 2024

```typescript
import { openai } from '@ai-sdk/openai';
import { generateText } from 'ai';

const { text } = await generateText({
  model: openai('gpt-4o'),
  prompt: 'Explica qué es RAG',
});
```
