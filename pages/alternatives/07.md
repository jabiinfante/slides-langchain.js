# OpenAI SDK + Custom Code

Usar directamente el SDK de OpenAI (o Anthropic, etc.)

<v-clicks>

### ¿Por qué ir "vanilla"?

- **Control total**: Sin abstracciones intermedias
- **Simplicidad**: Menos código, más claro
- **Performance**: Sin overhead de frameworks
- **Debugging**: Más fácil de debuggear

### ¿Cuándo NO usar frameworks?

✅ Aplicación simple con 1-2 llamadas a API
✅ Prototipo rápido
✅ Necesitas rendimiento máximo
❌ RAG complejo con múltiples fuentes
❌ Agentes con muchas herramientas
❌ Workflows complejos

### Instalación

```bash
npm install openai
```
