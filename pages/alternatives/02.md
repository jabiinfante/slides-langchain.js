# El Panorama JS/TS para LLMs

<v-clicks>

### Tres niveles de abstracción

```
┌─────────────────────────────────────────────────────┐
│  Frameworks (LangChain, LlamaIndex)                 │  ← Mayor abstracción
├─────────────────────────────────────────────────────┤
│  SDKs unificados (Vercel AI SDK)                    │  ← Abstracción media
├─────────────────────────────────────────────────────┤
│  APIs nativas (OpenAI SDK, Anthropic SDK, etc.)     │  ← Control total
└─────────────────────────────────────────────────────┘
```

### La pregunta no es "cuál es mejor"

Es: **¿qué nivel de abstracción necesito?**

- **Prototipo rápido** → API nativa o Vercel AI
- **RAG complejo** → LlamaIndex o LangChain
- **Agentes sofisticados** → LangChain o APIs nativas con más código
- **UI streaming en React** → Vercel AI SDK
