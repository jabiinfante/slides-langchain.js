# LlamaIndex

Framework especializado en indexación y retrieval (RAG)

<v-clicks>

### Características principales

- **Foco en RAG**: Diseñado específicamente para sistemas de búsqueda y retrieval
- **Data connectors**: Más de 160 conectores de datos
- **Índices sofisticados**: Tree, list, keyword, graph indexes
- **Query engines**: Múltiples estrategias de consulta

### ¿Cuándo usar LlamaIndex?

✅ Tu proyecto es principalmente RAG
✅ Necesitas conectores de datos diversos
✅ Quieres experimentar con índices avanzados
❌ Necesitas agentes complejos
❌ Workflows no relacionados con RAG

### Instalación

```bash
npm install llamaindex
```
