# RAG: Vector Stores

<v-clicks>

### Alternativas a MemoryVectorStore

| Vector Store | Características |
|--------------|-----------------|
| **PGVector** | PostgreSQL con extensión de vectores |
| **MongoDB Atlas** | Vector search en MongoDB cloud |
| **Milvus** | Open-source, grandes volúmenes |
| **Pinecone** | Managed, alta escalabilidad |
| **Weaviate** | GraphQL API, hybrid search |
| **Chroma** | Open-source, fácil desarrollo |
| **Qdrant** | Alto rendimiento, filtros avanzados |

### Ejemplo: PGVector

```typescript
import { PGVectorStore } from "@langchain/community/vectorstores/pgvector";

const vectorStore = await PGVectorStore.initialize(embeddings, {
  postgresConnectionOptions: { connectionString: process.env.DATABASE_URL },
  tableName: "documents",
});
```
