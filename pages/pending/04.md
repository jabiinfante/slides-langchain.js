# Tipos de datos no cubiertos

<v-clicks>

### Document Loaders especializados
- **Tablas**: Excel, CSV, SQL databases
- **Web scraping**: Crawling avanzado de sitios web
- **APIs**: Conectar con APIs REST/GraphQL
- **Multimedia**: Transcripción de audio, análisis de imágenes
- **Structured data**: JSON, XML, YAML

### Vector Stores adicionales
- **pgvector**: PostgreSQL con vectores
- **Milvus**: Para grandes volúmenes
- **Weaviate**: Con capacidades de GraphQL
- **MongoDB Atlas**: Vector search en MongoDB
- **Redis**: Para casos de uso específicos

### Modelos especializados
- **Vision models**: Análisis de imágenes con GPT-4V, Claude 3
- **Multimodal**: Combinar texto, imagen, audio
- **Code generation**: Modelos especializados en código
- **Domain-specific**: Modelos específicos de dominio (medicina, legal, etc.)

</v-clicks>
