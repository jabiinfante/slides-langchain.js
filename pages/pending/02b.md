# Deep Agents

<v-clicks>

### Librería standalone para agentes complejos

Inspirada en Claude Code, Deep Research y Manus.

### Características principales

- **Planificación**: Capacidad de planificar tareas multi-paso
- **File systems**: Sistema de archivos virtual para gestión de contexto
- **Subagentes**: Spawn de agentes especializados para subtareas
- **Long-term memory**: Integración con almacenamiento persistente

### Ejemplo básico

```typescript
import { createDeepAgent } from "@langchain/deepagents";

const agent = createDeepAgent({
  model: "gpt-4o",
  filesystem: true,
  planning: true,
});

await agent.invoke("Research and write a report about AI trends");
```
