# Testing y QA

<v-clicks>

### Testing de aplicaciones LLM
- **Unit testing**: Testear componentes individuales
- **Integration testing**: Testear flujos completos
- **Regression testing**: Detectar degradación de calidad
- **A/B testing**: Comparar diferentes prompts/modelos

### Evaluación automática
- **LLM-as-judge**: Usar LLMs para evaluar respuestas
- **Datasets de evaluación**: Crear y mantener datasets golden
- **Métricas custom**: Definir métricas específicas del dominio
- **Continuous evaluation**: Evaluación continua en producción

### Herramientas
- **LangSmith**: Testing y evaluación integrada
- **DeepEval**: Framework de testing específico
- **Promptfoo**: Testing de prompts
- **RAGAS**: Evaluación específica de RAG
