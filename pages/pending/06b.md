# Document Loaders: Ejemplos

<v-clicks>

### CSV Loader

```typescript
import { CSVLoader } from "@langchain/community/document_loaders/fs/csv";

const loader = new CSVLoader("data/sales.csv", {
  column: "description",
});
const docs = await loader.load();
```
