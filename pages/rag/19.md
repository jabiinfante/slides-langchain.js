# Dividir documentos (chunking)

<v-clicks>

Los documentos largos deben dividirse en trozos más pequeños

```typescript
import { RecursiveCharacterTextSplitter } from "@langchain/textsplitters";

const textSplitter = new RecursiveCharacterTextSplitter({
  chunkSize: 1000,        // caracteres por chunk
  chunkOverlap: 200,      // solapamiento entre chunks
});

const chunks = await textSplitter.splitDocuments(docs);

console.log(`Dividido en ${chunks.length} chunks`);
console.log("Primer chunk:", chunks[0].pageContent);
```

### ¿Por qué dividir?

- Los embeddings funcionan mejor con textos cortos y cohesivos
- Enviamos solo contexto relevante al LLM (ahorramos tokens)
- Mejora la precisión de la búsqueda
