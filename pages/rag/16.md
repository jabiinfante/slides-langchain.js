# Qdrant - Búsqueda con score

```typescript
// Búsqueda con puntuación de similitud
const resultsWithScore = await vectorStore.similaritySearchWithScore(
  "embeddings y vectores",
  3
);

resultsWithScore.forEach(([doc, score], i) => {
  console.log(`${i + 1}. Score: ${score.toFixed(3)}`);
  console.log(`   ${doc.pageContent}`);
});
```
