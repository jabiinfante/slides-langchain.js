# Evaluación de sistemas RAG

<v-clicks>

### Métricas clave

**Retrieval (recuperación)**
- Precision@K: % de documentos relevantes en top K
- Recall@K: % de documentos relevantes encontrados
- MRR (Mean Reciprocal Rank): posición del primer resultado relevante

**Generation (generación)**
- Groundedness: ¿La respuesta se basa en el contexto?
- Relevance: ¿La respuesta responde la pregunta?
- Faithfulness: ¿No hay alucinaciones?

### Herramientas

- **LangSmith**: Evaluación y debugging
- **RAGAS**: Framework específico para evaluar RAG
- **DeepEval**: Testing de LLM applications
