# Qdrant - Configuración

<v-clicks>

### Instalación

```bash
npm install @langchain/qdrant @qdrant/js-client-rest
```

### Iniciar Qdrant con Docker

```bash
docker run -p 6333:6333 qdrant/qdrant
```

### O usar Qdrant Cloud

Crear una cuenta gratuita en [qdrant.tech](https://qdrant.tech)

- 1GB de almacenamiento gratis
- Clusters gestionados
- Sin necesidad de infraestructura
