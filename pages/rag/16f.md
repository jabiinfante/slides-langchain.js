# Otros Document Loaders útiles

<v-clicks>

### CSV
```typescript
import { CSVLoader } from "@langchain/community/document_loaders/fs/csv";
const loader = new CSVLoader("./datos/productos.csv");
```

### DOCX (Microsoft Word)
```typescript
import { DocxLoader } from "@langchain/community/document_loaders/fs/docx";
const loader = new DocxLoader("./docs/informe.docx");
```

### Texto plano y Markdown
```typescript

const loader = new TextLoader("./docs/README.md");
```

### JSON
```typescript
import { JSONLoader } from "@langchain/community/document_loaders/fs/json";
```
