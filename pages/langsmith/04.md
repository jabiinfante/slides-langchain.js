# Configuración de LangSmith

<div class="mt-8 text-sm">

<v-click>

**Paso 1:** Crear cuenta gratuita en [smith.langchain.com](https://smith.langchain.com)

</v-click>

<v-click>

**Paso 2:** Obtener API Key desde Settings → API Keys

</v-click>

<v-click>

**Paso 3:** Configurar variables de entorno

```bash
# .env
LANGCHAIN_TRACING_V2=true
LANGCHAIN_API_KEY=lsv2_pt_...
LANGCHAIN_PROJECT=mi-proyecto
```

</v-click>

<v-click>

<div class="mt-6 text-xs opacity-80">
✨ Automático: LangChain detecta las variables y envía trazas a LangSmith
</div>

</v-click>

</div>
